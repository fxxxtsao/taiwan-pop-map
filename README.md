# 台灣人口年齡地圖

互動式台灣人口/所得熱力圖——全台 368 鄉鎮市區、7,701 村里雙層級，可依年齡層篩選人口分布，或切換綜稅所得（中位數/平均數）主題，並疊加全國國小與幼兒園位置。純靜態網頁（MapLibre GL JS），無後端。

## 快速開始

**本機預覽**

```bash
python -m http.server 8123 --directory web
# 開 http://localhost:8123
```

**Docker**

```bash
docker build -t taiwan-pop-map .
docker run -d -p 8080:80 taiwan-pop-map
# 開 http://localhost:8080
```

## 功能

- 人口熱力圖：總人口與各年齡層（0–100 歲自訂區間），人口數或占縣市比例兩種呈現
- 綜稅所得熱力圖：財政部村里統計（111 年度），中位數/平均數
- 縮放至村里層級自動切換（lazy load）
- 學校圖層：國小 2,582 所（藍）、幼兒園 6,546 所（琥珀）、同址附幼（紫），選定縣市/鄉鎮後以大頭針呈現
- 道路底圖：內政部國土測繪中心 EMAP（免金鑰）
