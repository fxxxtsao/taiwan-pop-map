# 移機部署 — 新主機從零到網頁運行

適用：把本專案移植到另一台主機，用 Docker 跑起來對外服務。
（部署原理與非 Docker 方案見 `部署指南.md`）

---

## 1. 前置安裝（一次性）

| 環境 | 動作 |
|------|------|
| Windows | 安裝 [Docker Desktop](https://www.docker.com/products/docker-desktop/)。**裝完須重開機**（WSL2），首次啟動要在它的視窗接受服務條款、等左下角引擎變綠 |
| Linux | `curl -fsSL https://get.docker.com | sh`，然後 `sudo usermod -aG docker $USER` 後重新登入 |
| 共通 | 安裝 git（用 clone 取得專案的話） |

驗證：`docker info` 有輸出、不報錯，才繼續。

## 2. 取得專案

```bash
git clone https://github.com/fxxxtsao/taiwan-pop-map.git
cd taiwan-pop-map
```

- repo 已含全部網站資料（`web/`），**不需要**跑 `scripts/` 資料管線、不需要 Node.js、不需要 `npm install`
- 無法連外的內網主機：改用「離線搬運」（見第 6 節）

## 3. Build 與啟動

```bash
docker build -t taiwan-pop-map .
docker run -d --name popmap -p 8080:80 --restart unless-stopped taiwan-pop-map
```

- `-p 8080:80`：左邊 8080 是對外 port（可自行改），右邊 80 固定（容器內 nginx）
- `--restart unless-stopped`：主機重開機後容器自動起來
- port 被占用會報 `port is already allocated` → 換左邊數字重跑

## 4. 驗證（依序做完才算部署成功）

```bash
docker ps          # STATUS 應為 Up，PORTS 顯示 0.0.0.0:8080->80/tcp
```

瀏覽器開 `http://localhost:8080`，逐項檢查：

1. 台灣地圖著色出現（不是全灰——全灰 = JSON 沒載到）
2. 道路底圖出現（主機需能連 `wmts.nlsc.gov.tw`；使用者瀏覽器也要能連 CDN，見 `部署指南.md` §7）
3. 滾輪放大，村里界出現（驗證大 JSON lazy load）
4. 切「綜稅所得」主題，地圖重新著色、圖例顯示年度
5. 下拉選一個縣市，大頭針落下
6. （可選）`curl -sI -H "Accept-Encoding: gzip" http://localhost:8080/population.json` 應含 `Content-Encoding: gzip`

## 5. 開放給其他人連線

- 同仁改用 `http://<主機IP>:8080` 連線
- Windows 主機要開防火牆：`New-NetFirewallRule -DisplayName "popmap" -Direction Inbound -Protocol TCP -LocalPort 8080 -Action Allow`（系統管理員 PowerShell）
- Linux 視發行版開 firewalld/ufw 對應 port
- 若只是給同仁「看」，不必自架——直接用 GitHub Pages：https://fxxxtsao.github.io/taiwan-pop-map/

## 6. 內網/離線主機（無法 git clone 或 docker pull 時）

在有網路的機器上打包 image，檔案搬過去載入：

```bash
# 有網路的機器
docker build -t taiwan-pop-map .
docker save taiwan-pop-map -o taiwan-pop-map.tar   # 約 25MB

# 離線主機（搬入 tar 後）
docker load -i taiwan-pop-map.tar
docker run -d --name popmap -p 8080:80 --restart unless-stopped taiwan-pop-map
```

注意：離線環境下瀏覽器連不到 CDN/字型/底圖，功能限制見 `部署指南.md` §7 的替代方案。

## 7. 日後更新版本

```bash
git pull                              # 拉最新版（含年度資料更新）
docker build -t taiwan-pop-map .      # 重新 build
docker rm -f popmap                   # 移除舊容器
docker run -d --name popmap -p 8080:80 --restart unless-stopped taiwan-pop-map
```

零停機需求不高的內部工具，這四行就是完整更新流程；資料本身的年度更新（抓新人口/所得/名錄）見 `規格書.md` §3.4，在任一台有 Node.js 的機器做完 commit + push 即可。
